Help Page!

Click Me!

Click on the boxed titles to open up sections of the help menu.
Each section has a description of the command, followed by example usages.
The usage would be used as if you did "@botsly [usage line]". I.E.

Usage:
  help

Would indicate you should do "@botsly help" to use the command in that way

Commands: Names

  • Nicknames: Get a list of your nicknames, everyone's nicknames, or set anyone's nickname
    • Usage:

    all nicknames
    @person's nicknames
    my nickname is Late for Dinner
    person's alias is Shirley

  • ID: Get your "GroupMe ID" for whatever reason. (For reals, only Daniel would use this)
    • Usage:

    my id
    @person's id

Commands: Address

  • Address: Set your own address. If you start with "college" you can set your college address. If there is nothing after "address" you can ask for someone's address or college address.
    • Usage:

    my address is 100 N. Main Street
    @personGuy's college address is 101 N. Main Street
    my address
    @someone's college address

Commands: Vacation

  • Vacation: Set a vacation for yourself or someone else. They are of the form "[person] will be going on vacation to [place] from [start time] to [end time]" but the search is pretty lenient for what words you use.
    • Usage:

    I'm going on vacation to Cancun from today until Saturday (will do today's date through the Saturday of this week)

    Joe is going on vacation to 100 E. Jerry Way from July 3rd through August 17th

  • Vacations: See who's going where. You can see your own vacations, and a list of all vacations
    • Usage:

    All vacations

    My vacations

    Joe's vacations

  • Remove Vacation: You can remove vacations if you are not going on them anymore. If you do "my vacations", each vacation has a number. Use that number to remove vacations
    • Usage:

    Remove vacation 1

    Remove my vacation 2

    Remove Joe's vacation 1

Commands: Analytics

  • Stats: Get stats on how many messages you post, how many per day, messages in a row, etc. Use "all" to get it for everyone
    • Usage:

    my stats
    @someone's stats
    all stats

  • Leaderboard: See who posts the most messages per day :) (Note it only counts days where at least one message has been posted)
    • Usage:

    leaderboard
