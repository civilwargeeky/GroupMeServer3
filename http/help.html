Help Page!

Click Me!

Click on the boxed titles to open up sections of the help menu.
Each section has a description of the command, followed by example usages.
The usage would be used as if you did "@botsly [usage line]". I.E.

Usage:
  help

Would indicate you should do "@botsly help" to use the command in that way

Commands: Names

  • Name: Set a name for someone. If you specify "real" it will set it as their real name
    • Usage:

    my name is Bill

    @JerryTron49's real name is Jerry

  • Names: List names. If you specify "all" it will print out all names for everyone
    • Usage:

    my names

    Jerry's names

    all names

  • ID: Get your "GroupMe ID" for whatever reason.
    • Usage:

    my id
    @person's id

Commands: Address

  • Address: Set an address. If you start with "college" you can set your college address. If there is nothing after "address" you can ask for someone's address or college address.
    • Usage:

    my address is 100 N. Main Street
    @personGuy's college address is 101 N. Main Street
    my address
    @someone's college address

  • All Addresses: Returns all addresses for everyone. Can specify with "college" or nah
    • Usage:

    All addresses

    All college addresses

Commands: Fun

  • Joke: Get some jokes from some website! You can also specify the number, and category of joke. You can also request different types of jokes
    • Joke Categories:
      "haha"
      "signs"
      "nerd"
      "professional"
      "quotest"
      "lightbulb"
      "blonde"
      "laws"
    Joke Types:
      Pokemon Joke
      Squirrel Fact
      Bat Fact
      Turtle Fact
      Bird Fact
      Aww Fact
      Fun Fact
      Snek Pic
      Corgi Pic
    Usage:

    joke
    tell me a joke
    tell me some jokes
    tell me some nerd jokes
    tell me 6 blonde jokes
    tell me 2 jokes
    show me a snek pic
    let's get 5 aww facts up in here

  • Subscriptions! To every joke or fact, you can be subscribed or unsubscribed.
    If you are subscribed, a joke will be posted either every time you post a message (jokes) or every time your post has a keyword in it (facts)
    To unsubscribe from some you may have to ask nicely :)
    • Usage:

    subscribe to bat facts
    please unsubscribe me from bat facts :)

  • Types: You can request to see what types of jokes exist
    • Usage:

    joke types
    fact types

  • Human Affection: You can request either love or human affection from botsly and he is legally required to oblige
    • Usage:

    LOVE ME

    Provide human affection to Joe, you robotic rapscallion
